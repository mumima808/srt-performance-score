# SRT Performance Score

*An SRT file is not a subtitle. It is a performance score.*

## What this is

For 30 years, the SRT format has been called a "subtitle file."
This project proposes a different frame:

An SRT file — sequence, timecode, text — has exactly the structure
of a musical score. It tells a performer what to do, when, and for
how long. When AI characters enter the picture, this is not a
caption system. It is a staging instruction.

## The core insight

Audio / Story
↓ Vrew
SRT = Performance Score
↓ DaVinci Resolve / any timeline
Stage
↓ LLM + AI characters
Performance
Language structure first. Visuals mount onto it.
This inverts the traditional workflow — and opens it to everyone.
